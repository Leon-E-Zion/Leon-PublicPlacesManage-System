# torchreid
- 需要安装，且需要指定位置的包进行安装
- 直接 pip install torchreid 导致错误{基于Linux与Windows11都出现了相关错误}
- 正确命令 ： pip install https://github.com/KaiyangZhou/deep-person-reid/archive/master.zip
---
# pycocotools
- yolo部分需要使用该工具包，需要 pip install pycocotools
- 在 pip install pycocotools 前 注意，此工具包需要c++编译器进行编译；因此需要提前安装对应的命令行提示的工具包（可以先运行根据报错获取相关提示）
